## Getting Started

First, run the development server:

```bash
 switch to node version 18
 # nvm use 20
 Install dependencies
 # npm i
```

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Make sure your local mongoDB is Up

Open [http://localhost:5000](http://localhost:5000) with your browser to see the result.
